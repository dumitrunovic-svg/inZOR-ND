Quantum chemistry · inZOR-ND · Discrete search · Planned benchmark

Photochemical multi-geometry active space — robust CAS across conformations

Test whether inZOR-ND discovers a small, shared active space that remains valid along photochemically difficult coordinates (quasi-degeneracy / CI-like regions), compared to AVAS, NOON/MP2, and matched-budget random search. No RL / NN / backprop — same emergent population paradigm as the existing suite.

Status

Study design delivered; full numerical tables will appear here after runs complete. Experimental protocol, folder layout, and aggregation helper live in the ZOR tree: zor_active_space/photochem_multigeom/ (see EXPERIMENTAL_PLAN.md, benchmark_specs.json).

Central question

Can inZOR-ND find an active space that is small and stable across multiple geometries in regimes where gaps collapse and single-reference / naive orbital rules often fail?

Multi-geometry evaluation uses existing CASFitness(List[MolSpec]) — one MO pool, one mask evaluated on every geometry in the scan.

Methods & baselines

  • inZOR-ND — Population + ActiveSpaceEnv; discrete combinatorial search with cached CASSCF/SA-CASSCF evaluations.
  • NOON / MP2-based — same construction as in comparative benchmarks.
  • AVAS — PySCF AVAS subspace, fair CAS truncation as in existing scripts.
  • Random — matched evaluation budget vs inZOR-ND (or explicit ratio in tables).

Feasibility & scientific value

F feasibility: High — the engine and multi-geometry CASFitness already power N₂ PEC, butadiene, and ethylene SA drivers; remaining work is scan design, uniform JSON summaries, and HTML refresh.

Scientific value: Positive if inZOR-ND finds consistent active spaces where baselines fragment; a negative result (no shared small CAS) still bounds what emergent discrete search can guarantee in photochemical scans.
