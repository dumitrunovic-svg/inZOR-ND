Single entry point for H₂ gap diagnostics, the ethylene quasi-degeneracy ZOR scene, and the published inZOR-ND Cr₂ / N₂ / comparative suite.

Updated 2026-04-02 · inZORi tests catalogue

Scope

This chapter bundles (i) a minimal H₂ bond-stretch table comparing inZOR-ND vs NOON orbital choices for the SA-CASSCF(2,2) HOMO–LUMO gap, (ii) the ethylene quasi-degeneracy work in the open-source ZOR environment (evolutionary exploration with a quantum-chemistry probe, not the inZOR-ND discrete search engine), and (iii) the existing inZOR-ND active-space benchmarks (N₂, Cr₂, comparative). Distinctions between engines are stated explicitly so claims stay aligned with each codebase.

Note. inZOR-ND refers to the bio-adaptive discrete search used for CASSCF active-space selection. ZOR refers to the 2D/3D artificial-life environment and validation batches for the ethylene scene. They share thematic motivation (emergent / adaptive behaviour) but are different programs and fitness landscapes.
